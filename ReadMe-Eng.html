Inquiry modul

Product has instead of "Add to cart" button "Send Inquiry" button, which opens pop up window including inquiry form. Button display can be configured in administration and is based on a product type. When inquiry form is submitted an email including form data is sent. If user is already loged in, inquiry form is completed by itself. All inquiries can be seen in administration panel which is accessible through menu "Sales" - "Inquiry".

Installation

Copy all files in to the root of your Magento web shop. When asked if you want to overwrite the files, click "Yes to all".

Installation - Magento

Step 1:In menu "Catalog" (1.) submenu "Attributes"(2.) choose an option "Manage Attributes"(3.) - see printscreen here

Step 2:In search field(1.) type "product_buy_type" and press enter. When search results are displayed, click on the name of the attribute(2.) - see printscreen here

Step 3:Window with options opens up. Find "Visible in Product View Page on Front-end"(1.) and "Used in Product Listing"(2.) and set their values on "yes"- see printscreen here

Important! Depend od shop configuration, sometimes there are some problems with path. If pop up doesn't show when clicking to Inquiry button, open inquiry.js and in line 30 change url: currentUrl + "inquiry/index/showForm", into url: "example.com/inquiry/index/showForm", where example.com is your domain.
